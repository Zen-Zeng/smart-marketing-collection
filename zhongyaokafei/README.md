# 中药咖啡项目

## 项目简介
中药咖啡项目是一款融合传统中药与现代咖啡文化的创新产品展示平台。本项目旨在向用户介绍中药咖啡的独特配方、健康功效及品牌理念。

## 核心优势
- **传统与现代融合**：将千年中药文化与现代咖啡工艺完美结合
- **健康与美味并存**：保留咖啡醇香的同时融入中药保健功效
- **文化传承与创新**：推动中医药文化在现代生活方式中的传承与发展

## 部署说明
本项目采用GitHub Pages进行部署，通过GitHub Actions自动将代码部署到gh-pages分支。
