## **一、 品牌顶层设计 (Brand DNA)**

### **1.1 品牌原点：东方花养 × 现代疗愈**

我们不只是卖面膜，而是贩卖“**爱自己**”的东方哲学。

* **核心叙事**：“传承东方花养美学，将东方花养智慧凝于方寸之间。我们坚信，**爱自己，是女性最美的修行**。”  
* **Slogan**：**以花养肤，宠爱如初**。  
* **差异化定位**：  
  * **视觉锤**：新中式宫廷美学 vs 工业风成分党。  
  * **情绪钩子**：不仅是护肤，更是“小主般”的仪式感与被宠爱感。

### **1.2 目标人群：情感驱动型消费者**

* **核心画像**：16-35岁，全国范围。  
* **消费心理**：  
  * 她们可能是 *Fan Beauty* 或 *敷尔佳* 的受众，但厌倦了单纯的功效堆砌。  
  * 她们愿意为“颜值、故事、明星同款”支付溢价，追求\*\*“身心疗愈”\*\*。

### **1.3 明星IP终局定位**

* **角色**：**首席体验官 (Chief Experience Officer)**  
  * *拒绝*：只挂名不试用的“割韭菜”代言。  
  * *坚持*：**“第一位挑剔的用户”**。所有产品必须明星亲自试用过关，且在Vlog中真实露出的。

### **1.4 市场竞对锚定 (Market Benchmarking)**

通过对标行业头部与明星IP品牌，精准寻找“仙小主”的生态位突围点。

| 竞对品牌 | 品牌定位 (Type) | 核心单品 & 价格带 | 核心优势 (Strength) | 仙小主破局点 (Our Opportunity) |
| :---- | :---- | :---- | :---- | :---- |
| **敷尔佳 (Voolga)** | **院线功能型** 医美修护心智 | 白膜/黑膜 ¥128-148/盒 | **硬核背书**：强大的医疗器械背景，消费者信任度极高。 | **补足“情绪价值”**：敷尔佳偏“药”感，体验冰冷；我们主打\*\*“花养香氛+愉悦肤感”\*\*，提供不仅有效更“好用”的体验。 |
| Fan Beauty Diary | 明星超级IP (范冰冰) | 海葡萄面膜 ¥138-198/盒 | 强人设绑定：范爷“美妆狂魔”人设深入人心，带货力极强。 | 差异化“东方叙事”：避开“现代女王”风格，深耕\*\*“东方宫廷美学”\*\*；强调“疗愈”而非单纯的“变美”。 |
| Code Mint | 红人IP+资本 (周扬青+雅诗兰黛) | 冰美式面膜 ¥100-300/盒 | 纯净美妆 (Clean)：主打“酷/飒/清醒”，视觉风格极简冷淡。 | 大众共情 (Mass)：Code Mint偏小众高冷；我们主打\*\*“温润/宠爱”\*\*，更符合大多数中国女性对“被呵护”的心理需求。 |
| UIR | 小众功效型 成分党 | 涂抹面膜 ¥100-200 | 极简高效：单一成分打透，针对性强。 | 流量降维打击：利用\*\*“明星流量+圈层资源”\*\*快速拉升声量，用品牌故事赋能冷冰冰的成分。 |

* **差异化突围总结**：  
  * **比功能型品牌（敷尔佳/UIR）更懂“美”**：不仅解决肌肤问题，更解决情绪问题（仪式感）。  
  * **比IP型品牌（Fan Beauty/Code Mint）更具“文化感”**：不依赖单一明星人设，而是将明星融入“东方花养”的宏大文化叙事中，延长品牌生命周期。

## **二、 产品战略：技术与矩阵 (Product)**

### **2.1 超级单品逻辑：花萃发酵系列**

* **核心技术壁垒 (RTB)**：**“独创花+酵+油三位护肤”**。  
  * **痛点**：传统油敷膜太油腻，纯水膜不够润。  
  * **方案**：利用**独家发酵专利** \+ **微乳化技术**，将花油发酵为小分子，实现“以油养肤，润而不腻”。  
  * **信任背书**：与 **Fan Beauty/敷尔佳** 同源大厂代工，共享大牌品控。

### **2.2 产品矩阵 (SKU Matrix)**

* **核心品类**：功能型花面膜（6款）  
  * 保湿 / 美白 / 抗衰老 / 水油平衡 / 舒缓 / 消肿  
* **组合策略**：  
  * **流量款**：9.9元/19.9元体验装（单片），用于直播间拉新。  
  * **利润款（主推）**：**周期护理礼盒**（如“7天花养急救包”），提升客单价，覆盖投放成本。

## **三、 方法论：EPOCH-Y 智能运营系统**

我们不把代运营看作简单的“卖货”，而是为品牌建立一个**自我进化的商业系统**。

### **3.1 核心模型 (The Model)**

* **E (Environment)**：洞察“东方美学复兴”与“女性悦己”的大趋势。  
* **P (Persona)**：明星人设与品牌人格的深度绑定（明星即品牌，品牌即明星）。  
* **O (Operating System)**：**内容飞轮 \+ 渠道飞轮** 双驱动。  
* **C (Capital)**：投放、资源与明星影响力的协同杠杆。  
* **H (Habitual Experiments)**：赛马机制，快速测试素材与人群包。

### **3.2 流量打法：1+N+X 矩阵**

* **1 (Head)**：**明星本人**。  
  * 动作：发布“剧组自用”Vlog，发起“半个娱乐圈都在用”的送礼开箱挑战。

  * 动作：美妆博主深度评测“花+酵+油”成分，建立专业口碑。

### **4.1 核心财务指标 (KPIs)**

* **首年GMV目标**：**2000 万元**

  * *策略*：不盲目烧钱打排名，通过高客单价套装和精准明星粉转化，确保每一分钱都有产出。

| 阶段 | 时间周期 | 核心目标 | 关键动作 |

## **五、 商业模式与合作方式 (Business Model)**

### **5.1 合作取费模型**

采用 **“基础服务费 \+ GMV增量分润”** 的共赢模式。

| 费用模块 | 结算方式 | 说明 |
| :---- | :---- | :---- |
| **基础服务费** | 固定费用 (按季预付) | 覆盖核心运营团队（项目经理/内容/直播）的基础人力成本。 |
| **销售佣金** | 按月GMV阶梯结算 | 设定 **8% \- 15%** 的阶梯佣金，业绩越高，比例越高。 |
| **投放资金** | 品牌方独立承担 | 双方共同设定ROI阈值（如1:2.5），数据透明，按实结算。 |

### **5.2 阶梯分润与激励机制**

为了激励运营团队不断挑战高目标，建议采用以下阶梯：

* **起步阶梯** (GMV \< 100W/月)：佣金 **8%**  
* **成长阶梯** (GMV 100W \- 300W/月)：佣金 **10%**  
* **爆发阶梯** (GMV \> 300W/月)：佣金 **12%**  
* **对赌激励**：若季度GMV超额完成 **120%**，超出目标部分的佣金比例提升至 **15%**，作为团队年终奖金池。

*注：所有结算均提供第三方后台（抖音/小红书/天猫）真实数据报表，确保财务透明。*
